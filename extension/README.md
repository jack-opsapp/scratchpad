# Slate Chrome Extension

Save notes to Slate from any webpage.

## Features
- Right-click selected text → "Save to Slate"
- Click extension icon to type a quick note
- Auto-fills with selected text on the current page
- Choose page/section destination
- Add tags
- Remembers your default page/section
- Ctrl+Enter to save quickly

## Setup

### Generate Icons
1. Open `icons/generate.html` in Chrome
2. Right-click each canvas → "Save image as..."
3. Save as `icon16.png`, `icon48.png`, `icon128.png` in the `icons/` folder

### Install Extension
1. Open Chrome → `chrome://extensions/`
2. Enable "Developer mode" (top right toggle)
3. Click "Load unpacked"
4. Select this `extension/` folder
5. The Slate icon appears in your toolbar

### Sign In
1. Click the extension icon
2. Enter your Slate email and password
3. You're ready to save notes!

## Usage
- **Quick note**: Click icon → type → select page/section → Save
- **Save selection**: Select text on any page → right-click → "Save to Slate"
- **Keyboard**: Ctrl+Enter to save from the popup
